## Description:

This is a backend part for diploma project developed, using Node.js/Express.js, MongoDB, Mongoose.js

## Run Locally

Clone the project

```bash
  git clone ––––
```

Go to the project directory

```bash
  cd <project directory>
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
